# CSE6350 NoSQL Repository

This repository contains coursework and reference material for CSE6350 focused on NoSQL storage systems, with an emphasis on LevelDB and WiscKey.

## Repository Contents

- `Lab 2 Report.pdf` - lab submission/report for the course
- `Lab2_WiscKey_Report.pdf` - report covering WiscKey and LevelDB concepts
- `leveldb-main/` or LevelDB source files - reference code used for study, reading, and experiments
- supporting documents and assets related to the course project work

## Topics Covered

- NoSQL storage systems
- key-value stores
- LevelDB architecture
- WiscKey design
- storage-engine tradeoffs

## Upstream Reference

The LevelDB source included here is based on Google's LevelDB project:
